## CÓDIGOS BINARIOS.

En los códigos binarios, la base del código es 2, estos son los más utilizados por los sistemas digitales. Un ejemplo es el código binario natural.
![[Pasted image 20231002143659.png]]
Con n cifras binarias, se pueden obtener $2{^n}$ combinaciones diferentes. 
Es decir que:  
+ Un grupo de *n* bits binarios es un código de *n* bits que puede asumir $2{^n}$ combinaciones distintas. Para que el código sea útil no se puede asigna ninguna combinación a más de un elemento y varían de 0 a $(2^{n}-1)$
+ Aunque el mínimo números de bits necesarios para escribir un código con $2{^n}$ elementos es distinto de *n*, el número máximo no está especificado. 
+ Con estas combinaciones podríamos tener $2{^n}!$ códigos distintos. 

**<u>Códigos binarios continuos:</u>** Son aquellos donde todas sus combinaciones son adyacentes, es decir difieren de un bit.
**<u>Códigos binarios cíclicos:</u>** Son aquellos donde la última combinación es adyacente a la primera.
