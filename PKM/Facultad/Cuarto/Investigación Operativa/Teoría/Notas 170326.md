# U1 - Programación Lineal
- Utilizamos modelos matemáticos para tomar decisiones luego de validarlos.
- La programación lineal son un conjunto de técnicas de búsquedas eficientes para optimizar modelos lienales restringidos.
- Un programa lineal tiene siempre una función objetivo (que es la que debe maximizar o minimazar) y restricciones.
- Toda las funciones que forman parte del sistema son todas funciones lineales.
- Variables de decision: Elementos sobre los cuales puedo interceder o cambiar.
- Funcion objetivo: Función que incluye las variables de decision, aunque quizás no todas formen parte de la solución óptima.
- Restricciones: Limitaciones impuestas al grupo de decisiones permisibles.
- Coeficiente: Son los valores que acompañan a mis variables de decisión y que contribuyen a la solución óptima.
- Solución óptima: Valores que toman las variables de la funcioń objetivo para optimizar mi problema.
- Óptimo: Es el valor que optimiza mi problema de PL.
- Formulación de un problema de Programación Lineal
	- Paso 1 - Enunciado del Objetivo: Exprese con palabras el objetivo y la función objetivo para la medición de su desempeño.
	- Paso 2 - Identifico las variables de decisión: Identificar verbalmente las variables de decisión tenien en claro qué decisiones deben toamrse para optimizar la función objetivo.
	- Paso 3 - Identifico las restricciones: Exprese cada restricción con palabras. Pueden ser >=, <= o =
	- Paso 4 -  Determino la Función Objetivo: Iventar una notación simbólica para las variables de decisión y exprese la función objetivo con símbolos y además cada restricción con símbolos.
- Siempre vamos a maximizar o minimizar.
- Clasificación de los modelos lineales:
	1. Modelo de Red: Modelos en los cuales intervienen orígenes y destinos. Son de las aplicaciones más importantes de la logística y distribución.
		- Modelo de Transporte: Se utiliza para determinar la manera de asignar los productos de sus diferentes almacenes a sus clientes, a fin de satisfacer la demanda con el menor costo de transporte posible.
		- Modelo de Asignación: Sirve para determinar la asignación óptima de recursos.
	2. Modelo de Planificación de la Producción: Se utiliza para la cantidad óptima a producir con el fin de maximizar ganancias o bien minimizar costos.
	3. Modelo de Marketing: Está pensado para el diseño de una campaña eficaz de publicidad.
